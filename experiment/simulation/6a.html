Analysis and Synthesis of Boolean Relations using Digital Comparator of two binary numbers

INSTRUCTION

  1. Apply high to \(V_{CC}\) and low voltage to ground(GND)
  2. For all the combinations of the 4 bit inputs A ( \(A_3 A_2 A_1 A_0\)),B ( \(B_3 B_2 B_1 B_0\)) verify that the LEDs are ON or not.
  3. Enter binary input A=0110 (Decimal value is 6), B=0110 (Decimal value is 6). Output A=B, the led will be on.
  4. Enter binary input A=1010 (Decimal value is 10), B=0010 (Decimal value is 2). Output A>B, the led will be on.
  5. Enter binary input A=0010 (Decimal value is 2), B=1010 (Decimal value is 10). Output A< B, the led will be on.
  6. Enter other input combination of A and B
  7. Note: Red button symbolize as Low (L), Green button symbolize as High(H).
